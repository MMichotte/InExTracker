# InExTracker

InEwTracker is an Income and Expense Tracker web-app. 

### Features :
- [ ] Add monthly income (can add multiple)
  - [ ] Amount 
  - [ ] Payement date 
- [ ] Add single income 
- [ ] Add monthly expense (can add multiple)
- [ ] Add single expense 
- [ ] Display balance for current month
  - [ ] define day of month to start 
- [ ] Display analytics
